# 💀 AmirScript

**بهترین اسکریپت‌های فارسی برای Roblox | بهترین برای ایرانیان**

---

## 🧠 ما کی هستیم؟

یه تیم کوچیک ایرانی هستیم که فقط یه هدف داریم: **گند زدن به بازی بدون اینکه کسی بفهمه.**

ما اسکریپت‌ها رو طوری می‌نویسیم که:
- ✅ **نامحسوس باشن** — نه باگ دارن، نه فریز می‌کنن، نه لو می‌دن
- ✅ **آن‌دیتکت باشن** — (تا جایی که میشه) از دید آنتی‌چیت مخفی می‌مونن
- ✅ **فان باشن** — چون قراره حال کنیم، نه اینکه استرس بن بگیریم

---

## 🚀 چه چیزی داریم؟

| اسکریپت | توضیح | وضعیت |
|---------|-------|-------|
| **ESP** | دیدن دشمن‌ها از پشت دیوار با رنگ‌های مختلف | ✅ آماده |
| **Aimbot** | قفل خودکار روی هدف با دقت بالا | ✅ آماده |
| **Triggerbot** | شلیک خودکار وقتی نشونه روی هدفه | ✅ آماده |
| **Target Lock** | قفل روی یه نفر خاص (برای اذیت کردن هکرهای دیگه 😈) | ✅ آماده |
| **Team Check** | تشخیص تیم خودی از دشمن | ✅ آماده |
| **GUI همیشه بالا** | منو حتی روی منوی Roblox هم باز می‌مونه | ✅ آماده |

## 📥 نصب و استفاده

1. یه **Executor** خوب دانلود کن (Solara, Delta, Fluxus)
2. اسکریپت رو کپی کن
3. توی Executor پیست کن
4. **Execute** رو بزن
5. با **Right CTRL** منو رو باز و بسته کن
6. حالشو ببر 🎮

---

## 🔧 ویژگی‌های فنی

- **Team Check** با مقایسه اسم تیم — خطای تشخیص صفر
- **Wall Check** با Raycast — دشمن پشت دیوار رو زرد نشون میده
- **Aimbot** بدون Lag — مستقیم از CFrame استفاده می‌کنه
- **Triggerbot** با VirtualInputManager — شلیک نرم و طبیعی
